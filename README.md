# Sistema de Gestión Logística

Aplicación web orientada a la administración de una empresa de logística. Permite gestionar viajes, camiones, empleados y cargas, facilitando la organización de tareas y el seguimiento de operaciones.

## Funcionalidades principales

- Gestión de viajes (alta, baja y modificación)
- Administración de camiones y control de estado (disponible / en reparación)
- Gestión de empleados y asignación de tareas
- Sistema de usuarios con autenticación (login con usuario y contraseña)
- Visualización de trabajos asignados (día, horario, ubicación, camión y carga)
- Registro de cargas con cantidad, peso y destino
- Control de errores mediante modificación o eliminación de registros

## Tipos de usuario

### 👤 Empleados
- Acceso a sus tareas asignadas
- Visualización de horarios, ubicación y recursos asignados

### 🛠️ Administrador
- Creación y gestión de viajes, empleados y camiones
- Administración de cargas
- Modificación de estados y eliminación de registros

## Tecnologías utilizadas

- PHP
- MySQL
- JavaScript
- HTML/CSS
